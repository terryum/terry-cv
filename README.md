# terry-cv

Academic CV of **Terry Taewoong Um** (엄태웅), generated as Korean & English PDFs from a single Korean markdown source.

- Edit **`cv.ko.md`** only.
- Run `npm install` once, then `npm run build` → `dist/cv.ko.pdf` & `dist/cv.en.pdf`.
- Design: Classic Academic (serif, single column). Rendered via HTML/CSS + headless Chrome.

See [`CLAUDE.md`](./CLAUDE.md) for the full editing/build workflow.
